# Guardian Dashboard

Guardian Dashboard website build with Vue 3, Typescript, Vite, Windi CSS, Vuex

## Requirements

- node 14+
- npm 7+

## Getting started

1. Install dependencies:

`npm install`

2. Start developing an application:

`npm run serve`

3. Lint the code:

`npm run lint`

4. Run unit tests:

`npx vitest`

6. Build package:

`npm run build`

7. To see how it looks on production:

`npm run docker.build`
`npm run docker.run`
